O que é?

O C3 CEP é um buscador de CEP que tem como prioridade auxiliar na busca de CEPs em todo Brasil. Nós utilizamos o Web-service gratuito ViaCep para fazer consultas à base de dados.

Integramos a API de geocodificação direta PositionStack para fazer a conversão do endereço retornado pelo VIACEP em latitude e longitude.

E por fim, com as variáveis de latitude e longitude setadas, conseguimos usar a Maps JavaScript API da Google para mostrar a localização no mapa.

O servidor foi feito em Node.js.
